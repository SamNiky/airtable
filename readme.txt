Данный проект представляет собой простой CLI скрипт для работы с таблицой сервиса Airtable,
и выводом данных в кабинет веб-приложения основанного на Django.
Инструкция по началу работы:

1. Из директории /airtable
Выполнить команду -$ pip install -r requirements.txt
для установки пакетов необходимых в работе.

2. Перейти в директорию /airtable/API-Airtable
Выполнить команду -$ pip install .
Для установки CLI скрипта

3. Для подключения к свой БД и API Airtable необходимо совершить настройку в файле settings.py в этой же директории.
По умолчанию работает с таблицой https://airtable.com/shrTC1UxVE4hDdxTR и моей БД.

4. Для запуска скрипта нобходимо выполнить команду -$ runairtable
Скрипт самостоятельно возьмет все данные из Airtable и запишет их в базу данных,
если запись была изменена, изменит ее в БД, или удалит, если она была удалена из таблицы.

5. Для запуска Django локально, необходимо перейти в директорию /airtable/psychotherapist
И выполнить команду -$ python manage.py runserver

Запуск скрипта во время работы сервера выведет данные на сайт
